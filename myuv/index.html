The purpose of this site is for movie-watchers to quickly find and compare movie ratings all across the web. My name is Elfrey Shira, and I created this site. Before choosing to watch a movie, I always search the online ratings. I want to know whether a movie is worth watching. Additionally, I am a novice web-developer, and I wanted to learn how to make API calls with JSON and all that jazz. I chose to title it "Myuv" (pronounced m'yoov) because it sounded like a shorter and hipper version of "movie". So go ahead and search for that movie that you think you might want to watch but are not sure how good it is. Ratings are provided by reputable sources from IMDB, Rotten Tomatoes, and The Movie DB.

About the site, me, and you.
